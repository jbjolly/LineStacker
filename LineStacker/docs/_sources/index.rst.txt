Welcome to LineStacker's documentation!
===================================

LineStacker is a new open access tool for stacking of spectral lines. LineStacker is an ensemble of both CASA tasks and native python tasks, and can stack both 3Dcubes or already extracted spectra. Additionaly a set of tools are included to help further analyse stacked spectra and stacked sample.

Some example, showing how to use of LineStacker, can be found in the example section.
